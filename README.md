# React Personal Portfolio

A responsive personal portfolio website built with React.js to showcase my work, skills, and contact information in a clean and modern layout.

## Live Demo

[View Live Site](https://gergesnashaat.vercel.app)

## Overview

This project is my personal portfolio website. It was originally inspired by a tutorial, but I customized it by changing sections, removing parts that did not fit my needs, and adding my own touches to better represent my style and work.

The website is designed to introduce who I am, highlight selected projects, and provide an easy way for visitors or recruiters to get in touch.

## Features

- Responsive modern portfolio layout
- Reusable React component structure
- Hero/banner section for personal introduction
- Projects showcase section
- Contact section
- Footer with additional links/info
- Smooth visual presentation with animations and icons
- Bootstrap-based responsive UI

## Tech Stack

- React.js
- JavaScript
- CSS3
- Bootstrap
- React Bootstrap
- Animate.css
- React Icons
